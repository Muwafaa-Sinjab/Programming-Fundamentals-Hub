# 18 - Browsers

## Overview

This video explains the role of web browsers in accessing the World Wide Web. It covers what a browser is, the process of loading a web page, and how browsers interpret HTML code to display visual content.

## Main Topics

### What is a Browser?

A browser is a software application used to access and view content on the World Wide Web. It provides a graphical interface that displays text, images, videos, and other media from websites. Some of the most common web browsers include Google Chrome, Mozilla Firefox, Microsoft Edge, and Safari.

### How a Browser Works

When a user types a website name into the address bar and presses Enter, the browser sends an HTTP Request to the server that hosts the website. The server receives this request, locates the required data, and sends a file back to the browser as an HTTP Response. From a technical perspective, loading a web page is essentially the browser downloading this response file.

### HTML and Rendering

The file sent by the server is typically written in HTML (HyperText Markup Language). HTML is not a programming language; it is a structural language that uses tags to organize and classify information, such as defining titles, paragraphs, and image locations. The browser acts as a translator that reads these HTML tags and renders them visually on the screen. By interpreting the code, the browser turns plain text instructions into a fully formatted web page with colors, fonts, and layouts.

## Key Takeaways

- A browser is a software program that provides access to the World Wide Web
- Browsers load web pages by sending HTTP requests to servers and downloading the response files
- Web pages are primarily structured using HTML, which uses tags to organize content
- The browser interprets HTML code and renders it as a visual web page for the user
