# 20 - What is Domain Name? From Where to Get One?

## Overview

This video explains the concept of domain names and their role in accessing websites. It covers how domain names function as readable alternatives to IP addresses, how they are structured, and the process of purchasing and registering a domain.

## Main Topics

### What is a Domain Name?

Every server on the internet has a numerical IP address, but these numbers are difficult for people to remember. A domain name is a readable text name that is used in place of an IP address. It functions like a contact name saved in a phone; instead of typing a long string of numbers, the user types a memorable name to reach the intended server. A domain name must be unique, meaning no two websites can have the exact same domain.

### Domain Name Structure

A domain name typically consists of two main parts:
- **Second-Level Domain**: The main chosen name of the website (e.g., "google").
- **Top-Level Domain (TLD)**: The extension that follows the name, such as .com, .org, .net, or country-specific codes like .eg.

Together, these form the complete domain name, such as "google.com".

### How Domains Work

When a user types a domain name into a browser, the browser uses a system called DNS to translate the text name into the correct IP address. The browser then uses this IP address to connect to the server and load the website. The domain itself is simply a bridge that makes this process easier for humans.

### Where to Get a Domain Name

Domain names are purchased from companies known as Domain Registrars. Examples of these registrars include websites like Namecheap. To get a domain, a user visits a registrar, types in their desired name, and selects an available extension like .com or .org.

### Domain Availability and Ownership

If a domain name has already been registered by someone else, the registrar will indicate that it is unavailable. In some cases, the current owner might be willing to sell the domain, and the registrar may provide an option to "Make an Offer" to purchase it. There is a market for people who buy domain names with the intention of reselling them later at a higher price. Additionally, certain extensions like .gov are restricted and reserved specifically for government use. Registrars often sell additional services alongside domains, such as SSL certificates, email hosting, and web servers.

## Key Takeaways

- A domain name is a human-readable text address that points to a server's numerical IP address
- It consists of a chosen name and a Top-Level Domain (TLD) such as .com or .org
- Domains are purchased from companies called Domain Registrars
- A domain must be unique, and if it is already taken, it might be available for purchase from the current owner
- Certain domain extensions like .gov are reserved for specific entities like governments
