# 17 - Internet vs WWW

## Overview

This video explains the difference between the Internet and the World Wide Web. It defines the Internet as a physical infrastructure and the Web as a collection of content, and clarifies how the two concepts relate to each other.

## Main Topics

### What is the Internet?

The Internet is a massive global network of connected networks and devices. No single person, company, or government owns it. It acts as the physical infrastructure that allows data to travel across the world. This infrastructure consists of underwater fiber optic cables, large data centers filled with servers, and powerful routers that pass data from one country to another. The Internet is the pathway used to transfer all types of data, including emails, video calls, files, and application data.

### What is the World Wide Web (WWW)?

The World Wide Web is a collection of websites and web pages linked together through hyperlinks. The Web is not a physical network; it is the content that users view, read, hear, and download. To function, the Web relies on three main components:
- A communication protocol, such as HTTP or HTTPS, to manage the transfer of data between browsers and servers.
- A URL, which is the address typed into the browser to locate a specific page.
- HTML, which is the language used to write and structure the content of the web pages.

### The Difference Between the Two

The main difference is that the Internet is the physical infrastructure, while the Web is a service that runs on top of that infrastructure. The Internet contains the World Wide Web, but they are not the same thing. Another key difference is their age. The Internet is much older and was originally developed for different purposes. The World Wide Web was not introduced until 1991, when it was created as a method to share and access information over the existing Internet network.

## Key Takeaways

- The Internet is a global physical infrastructure of cables, servers, and routers that no single entity owns
- The World Wide Web is a collection of linked web pages and websites accessed through the Internet
- The Web relies on HTTP or HTTPS protocols, URLs, and HTML to function
- The Internet is the infrastructure, while the Web is the content that travels over it
- The Internet is much older than the World Wide Web, which was created in 1991
