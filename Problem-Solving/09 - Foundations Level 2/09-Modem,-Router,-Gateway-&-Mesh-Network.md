# 09 - Modem, Router, Gateway & Mesh Network

## Overview

This video explains the hardware devices used to establish and distribute internet connections. It covers the specific roles of modems, routers, and gateways, and introduces mesh networks as a solution for covering large physical areas.

## Main Topics

### Modem

A modem is the device that connects a local setup to the Internet Service Provider (ISP) through cables, DSL, or fiber. Its primary function is to receive the signal from the ISP and convert it into a digital internet format. A basic modem only supports a single device connection, and that connection must be made using a cable.

### Router

A router connects to the modem and creates a local network. Its job is to take the digital internet from the modem and distribute it to multiple devices, either through additional cables or wirelessly. Routers typically operate on two frequency bands:
- **2.4 GHz**: Slower speed but covers a wider area and penetrates walls more easily.
- **5 GHz**: Faster speed but covers a shorter distance and is more affected by physical obstacles.

In many modern setups, the modem and the router are built into a single combined device.

### Gateway

A gateway performs a similar function to a router but is designed to connect networks that use different protocols. If devices on different networks need to communicate but do not speak the same protocol, the gateway translates the data between them. If all connected networks use the exact same protocol, a standard router is sufficient and a gateway is not required.

### Mesh Network

A mesh network is a system used to provide internet coverage across large areas where a single router cannot reach all rooms. It consists of a main router connected to the modem, along with several additional nodes or secondary routers placed around the building. These nodes connect to each other wirelessly to create a single, unified network. The main advantage of a mesh network is that there are no dead zones, and if one node fails, the others continue to function. However, because the nodes communicate wirelessly, the speed is generally lower than a fully wired setup.

## Key Takeaways

- A modem connects the local network to the ISP and converts the signal into digital internet
- A router creates a local network and distributes the internet to multiple devices
- Routers use 2.4 GHz for wider range and 5 GHz for faster speeds
- A gateway translates protocols between different networks, while a router connects networks using the same protocol
- A mesh network uses multiple connected nodes to eliminate dead zones in large areas
