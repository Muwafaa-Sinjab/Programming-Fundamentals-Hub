# 22 - Sub Domain

## Overview

This video explains the concept of subdomains. It covers what a subdomain is, how it is structured in relation to a main domain name, and how it is used to organize large websites or direct users to entirely different services.

## Main Topics

### What is a Subdomain?

A subdomain is an additional part added to the beginning of a main domain name. It is used to separate a large website into smaller, organized sections. Instead of grouping all content under a single main page, a subdomain acts as a distinct pathway for a specific type of content, making the website easier to manage and navigate.

### Structure

A subdomain is placed before the main domain name, followed by the Top-Level Domain (TLD). For example, in the address "blog.google.com", "blog" is the subdomain, "google" is the main domain name, and ".com" is the TLD. The subdomain essentially divides the main domain into smaller, focused sections.

### Practical Uses

Subdomains are commonly used to organize different areas of a business or service. For example, a company might use "shop.company.com" for its online store, "blog.company.com" for news and articles, and "admin.company.com" for the internal management panel. Each subdomain provides a dedicated space for its specific function while remaining logically connected to the primary domain.

### Server Independence and Cost

An important feature of a subdomain is that it does not have to point to the same server as the main domain. A company can direct a subdomain to a completely different server, application, or system. For instance, the blog subdomain might run on one server while the shop subdomain runs on another, even though both share the same main domain name. Additionally, creating subdomains usually does not require purchasing a new domain, and there is typically no extra cost to create them within an existing domain subscription.

## Key Takeaways

- A subdomain is a prefix added before the main domain name to create a separate section
- It helps organize large websites into manageable parts like blogs, shops, or admin panels
- Subdomains can be directed to completely different servers or applications than the main domain
- Creating subdomains is typically free and does not require purchasing a new domain name
