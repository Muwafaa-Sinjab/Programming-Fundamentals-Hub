# 26 - What is XML?

## Overview

This video introduces XML. It explains that XML is a markup language used for structuring and organizing data in a readable format, distinguishes it from programming languages, and highlights its common use cases in data exchange and system configuration.

## Main Topics

### What is XML?

XML stands for eXtensible Markup Language. It is a descriptive language used to organize data in a structured, tree-like format using tags. It is not a programming language because it does not contain programming logic or operations; it only holds and describes data. Because it is written in plain text, XML can be read by both humans and any operating system, making it a flexible way to store information much like a simple database file.

### How XML Works

XML uses tags to define and classify information, which is similar to how HTML works. However, there is a key difference. In HTML, the tags are fixed and predefined by the language. In XML, the developer creates custom tags based on the specific data being stored. For example, a file holding employee data can use custom tags like `<name>`, `<age>`, and `<job>`. The data is organized hierarchically, with a root element containing child elements for each record, making the structure clear and easy to follow.

### Practical Uses of XML

XML is widely used to exchange data between different software systems because its structure is easily understood by any application. It is also commonly used for application configuration files, such as `config.xml`. Additionally, XML is the underlying format for Microsoft Office documents like Word (.docx) and Excel (.xlsx) files, and it is used in older SOAP-based APIs. Using a related technology called XSLT, XML data can also be transformed into other visual formats like HTML or PDF.

## Key Takeaways

- XML stands for eXtensible Markup Language
- It is a descriptive language for structuring data, not a programming language
- XML allows developers to create custom tags to fit their data, unlike HTML
- It is widely used for data exchange between different systems and for configuration files
- XML can be transformed into other formats like HTML or PDF using XSLT
