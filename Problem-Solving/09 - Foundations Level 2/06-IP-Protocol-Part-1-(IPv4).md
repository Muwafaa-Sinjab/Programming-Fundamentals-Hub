# 06 - IP Protocol Part 1 (IPv4)

## Overview

This video introduces the concept of an IP address and explains the structure of IPv4. It covers how devices are uniquely identified on a network, the technical composition of an IPv4 address, and the total number of addresses this version can support.

## Main Topics

### IP Address

Any device connected to a network or the internet has a unique identifier called an IP address. It functions like a physical home address, allowing data to be routed to the correct location. Every device has a different IP address, making it unique to that specific machine, much like a human fingerprint. This address can usually be found in the device's network settings.

### Internet Protocol

IP stands for Internet Protocol. It is a set of rules that organizes how data is sent and received between devices. An important point is that the Internet Protocol works for any direct connection between two devices, even if they are not connected to the wider internet. IP and TCP are complementary protocols that work together to handle data transfer.

### IPv4 Structure

IPv4 stands for Internet Protocol version 4. It is the most common format for identifying devices. An IPv4 address is written as four separate numbers separated by dots. Each number in the address can range from 0 to 255. For example, an address might look like 192.168.1.1. Technically, each of these four numbers represents one byte, meaning an IPv4 address is made up of 4 bytes, which equals 32 bits in total.

### IPv4 Capacity

Because an IPv4 address is built on 32 bits, there is a mathematical limit to how many unique addresses can be created. The total number of unique IPv4 addresses available is approximately 4.2 billion. This number represents the maximum limit of devices that can be directly addressed using this specific version of the protocol.

## Key Takeaways

- An IP address is a unique numerical identifier for a device on a network
- IP stands for Internet Protocol, the rules that govern data transmission
- The Internet Protocol works for any connection between devices, not just the internet
- IPv4 consists of four numbers separated by dots, with each number ranging from 0 to 255
- An IPv4 address is 32 bits long, allowing for about 4.2 billion unique addresses
