# 27 - What is JSON?

## Overview

This video introduces JSON, a lightweight format used for structuring and organizing data. It explains how JSON differs from XML, its unique key-value structure, and why it has become the standard format for modern web applications and APIs.

## Main Topics

### What is JSON?

JSON stands for JavaScript Object Notation. Like XML, it is a method for structuring data so it can be easily read by both humans and machines. However, JSON is not a programming language; it is simply a plain text format for organizing information. It was designed to be a simpler and lighter alternative to XML.

### Structure and Syntax

JSON organizes data using "Key: Value" pairs. The key is always a text string enclosed in quotes, and the value can be a string, a number, or other data types. For example, a record might look like `"name": "Ahmed Ali"`. To group multiple records together, JSON uses arrays by placing square brackets around a list of these objects, making it easy to represent complex data like a list of employees.

### JSON vs XML

The main difference between the two formats is how they handle data labels. XML requires an opening and a closing tag for every piece of information, which adds extra text and increases the file size. JSON eliminates these repeated tags by using the key-value structure. Because it does not repeat labels, JSON produces smaller, lighter files that are faster to write, easier for humans to read, and faster for machines to process.

### Where JSON is Used

Because of its speed and lightweight nature, JSON has become the standard format for modern APIs and web applications. While XML is still used in specific areas, such as older SOAP APIs, Microsoft Office files (like .docx and .xlsx), and application configuration files, JSON is the preferred choice for almost all new systems and mobile applications.

## Key Takeaways

- JSON stands for JavaScript Object Notation and is a lightweight data format
- It structures data using "Key: Value" pairs instead of opening and closing tags
- JSON uses arrays to group multiple data records together
- It produces smaller and faster files compared to XML
- JSON is the standard format used in modern APIs and web applications
