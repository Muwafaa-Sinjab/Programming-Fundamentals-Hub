# 23 - URL (Uniform Resource Locator)

## Overview

This video explains the concept of a URL. It covers what a URL is, the specific parts that make up a complete web address, and how URLs are used to locate and access different resources on the internet.

## Main Topics

### What is a URL?

URL stands for Uniform Resource Locator. It is the full address used to locate and access any resource on a network or the internet. A resource can be a web page, an image, a video, a file, or an API. A URL serves two main purposes: it identifies the exact location of the resource and specifies the method used to access it.

### Components of a URL

A complete URL is divided into several parts. Using `https://www.google.com/index.html` as an example:

- **Protocol**: The method or rules used to access the resource, located at the very beginning of the address. Common examples include `http`, `https`, and `ftp`. The `https` protocol indicates that the connection is secure.
- **Domain Name**: The name of the website, which the DNS system translates into an IP address to locate the correct server. In this example, it is `www.google.com`.
- **Path**: The specific folder or file location on the server where the resource is stored. In this example, `/index.html` points directly to a specific HTML file.

### Additional URL Elements

Besides the main components, a URL can include extra elements to provide more specific instructions to the server:
- A question mark (`?`) is used to start a query string, which passes additional parameters or search data to the server.
- A hash symbol (`#`) is used to point to a specific section or fragment within a single web page, making the page scroll directly to that location.

### URLs and Resources

Every distinct resource on a website has its own unique URL. For example, the main page of a site has one URL, while the maps section, email section, and images section each have completely different URLs. Changing the path at the end of the address directs the browser to load a different resource from the same server.

## Key Takeaways

- URL stands for Uniform Resource Locator
- It provides both the location and the access method for any resource on the internet
- The main components of a URL are the protocol, domain name, and path
- Query strings and fragments can be added to pass data or point to specific sections of a page
- Every web page, image, or file on a server has its own unique URL
