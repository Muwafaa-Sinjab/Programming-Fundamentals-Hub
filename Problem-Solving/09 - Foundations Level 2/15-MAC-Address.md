# 15 - MAC Address

## Overview

This video introduces the MAC address. It explains what a MAC address is, how it is formatted, and the difference between using a MAC address and an IP address for device identification on a network.

## Main Topics

### Definition of MAC Address

MAC stands for Media Access Control. A MAC address is a unique hardware identifier assigned to a device's network interface card. It is embedded by the manufacturer at the factory and remains fixed to the hardware, much like a vehicle identification number. Every device that connects to a network, such as a computer, router, or printer, has its own unique MAC address.

### Address Format and Structure

A MAC address is 48 bits long and is written as six pairs of hexadecimal characters separated by colons. For example, it looks like 00:1A:2B:3C:4D:5E. The address is divided into two parts. The first three pairs identify the specific manufacturer of the network hardware. The last three pairs identify the exact device itself. Because of this structure, it is possible to determine the manufacturer of a device simply by looking at its MAC address.

### Usage in Local Networks

The MAC address is primarily used for communication within a Local Area Network (LAN). When data is transferred between devices on the same local network, the system uses the MAC address to identify and deliver data to the exact physical machine.

### MAC Address vs IP Address

While both addresses identify a device, they serve different purposes. The IP address is a logical address that determines where the device is located on the network, and it can change depending on the connection. The MAC address is a physical address that points to the actual hardware, and it never changes. When data travels across the internet to external networks, routers rely on IP addresses to direct the traffic. The MAC address is only used to deliver the data to the correct device once it arrives on the local network.

## Key Takeaways

- MAC stands for Media Access Control
- It is a permanent, unique hardware address assigned by the manufacturer
- The address is 48 bits long and written in hexadecimal format
- The first half of the address identifies the hardware manufacturer
- MAC addresses are used for device identification within a local network, while IP addresses are used for routing across the internet
