# What Is a Computer? Functions and Components

## Definition

A computer is an electronic device that takes in data, processes it through a set of instructions, and produces useful information as a result.

## The Five Core Functions

| Function   | Role                                  |
|------------|----------------------------------------|
| Input      | Receives data from input devices       |
| Processing | Converts raw data into information     |
| Output     | Presents the processed result          |
| Storage    | Keeps data and information for later   |
| Control    | Coordinates all the other functions    |

Data moves through the system in one direction: input devices capture it, the processor works on it, output devices display it, and storage keeps a copy.

## Software vs. Hardware

**Software** is the set of instructions that tell the computer what to do. It splits into:
- Operating systems (Windows, macOS, Linux) that manage the machine itself
- Application software (browsers, editors, games) that users interact with directly

**Hardware** is the physical equipment: monitor, keyboard, mouse, printer, disk drives, and so on.

## Main Hardware Units

| Unit               | Purpose            | Examples       |
|---------------------|---------------------|-----------------|
| Input units          | Bring data in       | Keyboard, mouse |
| Output units         | Show results        | Monitor, printer|
| Main memory          | Temporary storage   | RAM, ROM        |
| Secondary storage    | Permanent storage   | HDD, USB drive  |
| CPU                  | Processing/control  | Processor       |

## Inside the CPU

- **ALU (Arithmetic Logic Unit):** carries out arithmetic and logical operations
- **CU (Control Unit):** coordinates and directs the other components
- **Registers:** extremely fast, very small storage used during processing

## Memory Hierarchy

Fastest and smallest to slowest and largest:

```
Registers → Cache → RAM → Hard Disk
```

Each step down trades speed for capacity.

## GPU

The GPU is a dedicated processor for handling images and video, built into the graphics card. It matters most for gaming and graphics-heavy applications.

## Key Takeaway

A computer combines hardware (physical parts) and software (instructions) to receive, process, output, and store data, with the CPU coordinating the entire process.
