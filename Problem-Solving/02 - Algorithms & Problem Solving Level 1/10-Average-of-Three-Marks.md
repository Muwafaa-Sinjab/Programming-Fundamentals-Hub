# 10 - Average of Three Marks

## Problem Statement

Write a program to ask the user to enter Mark1, Mark2, and Mark3, then print the average of entered marks.

## Steps

**Step 1:** Ask the user to enter Mark1, Mark2, Mark3.

**Step 2:** Avg = (Mark1 + Mark2 + Mark3) / 3.

**Step 3:** Print Avg.
