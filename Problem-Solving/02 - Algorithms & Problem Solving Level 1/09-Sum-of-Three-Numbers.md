# 09 - Sum of Three Numbers

## Problem Statement

Write a program to ask the user to enter Number1, Number2, and Number3, then print the sum of entered numbers.

## Steps

**Step 1:** Ask the user to enter Num1, Num2, Num3.

**Step 2:** Sum = Num1 + Num2 + Num3.

**Step 3:** Print Sum
