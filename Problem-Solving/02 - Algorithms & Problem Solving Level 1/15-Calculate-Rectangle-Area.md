# 15 - Calculate Rectangle Area

## Problem Statement

Write a program to calculate rectangle area and print it on the screen.

## Steps

**Step 1:** Ask the user to enter a, b.

**Step 2:** Area = a * b.

**Step 3:** Print Area.
