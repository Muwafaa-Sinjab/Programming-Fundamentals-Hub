# 04 - Autos Window

## Overview

This video introduces the Autos Window in the debugging environment. It explains how this window automatically displays variable values during a debugging session without requiring the developer to hover over each variable individually.

## Main Topics

### The Autos Window

The Autos Window is a panel located at the bottom of the program interface. While the program is paused in debugging mode, this window automatically lists the variables from the current line of code and the lines immediately preceding it, showing their stored values.

### Opening the Autos Window

This window may appear automatically when a debugging session starts. If it does not open by default, it can be accessed manually through the top menu by selecting "Debug", then "Windows", and finally choosing "Autos".

## Key Takeaways

- The Autos Window displays variable values automatically during debugging.
- It removes the need to hover the mouse over variables to inspect their values.
- It can be opened manually via the Debug > Windows > Autos menu path.
