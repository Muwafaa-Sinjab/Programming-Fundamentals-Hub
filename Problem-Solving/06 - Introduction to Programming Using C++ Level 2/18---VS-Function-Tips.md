# 18 - VS Function Tips

## Overview

This video presents several useful shortcuts in Visual Studio for navigating and managing functions. These tools help speed up the development process by allowing quick access to function definitions, declarations, and references without manually searching through the code.

## Main Topics

### Minimizing Function Code

A minus sign appears next to the function type in the editor. Clicking this sign collapses the function body, hiding the internal code from view to make the file easier to read.

### Go to Definition (F12)

This shortcut moves the cursor directly to the location where the function is implemented. It takes the developer from the function call straight into the function body.

### Go to Declaration (Ctrl + F12)

This shortcut moves the cursor to the function declaration. It is useful for quickly viewing the function signature, return type, and parameters without scrolling to the actual definition.

### Find All References (Shift + F12)

This opens a window listing every instance in the project where the selected function is called. It helps track how and where a function is being used across multiple files.

### Peek Definition (Alt + F12)

This opens a small inline window displaying the function definition. It allows the developer to read the implementation code without leaving the current line or opening a new tab.

### Rename (Ctrl + R)

This shortcut renames the function and automatically updates every reference and call to that function throughout the entire project at the same time.

## Key Takeaways

- Visual Studio provides built-in shortcuts to navigate functions quickly.
- F12 and Ctrl + F12 are used to jump to the definition and declaration respectively.
- Shift + F12 locates all places where a function is called.
- Alt + F12 previews the definition without leaving the current code line.
- Ctrl + R safely renames a function and updates all its calls automatically.
